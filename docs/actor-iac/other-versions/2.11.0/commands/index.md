# Commands

actor-IaC provides the following CLI commands:

| Command | Description |
|---------|-------------|
| [run](./run) | Execute workflows on local or remote nodes |
| [list](./list) | List available workflows in a directory |
| [describe](./describe) | Display workflow structure and step descriptions |
| [log-search](./log-search) | Search and query workflow execution logs |
| [log-serve](./log-serve) | Start a centralized log server |
| [log-merge](./log-merge) | Merge multiple log databases into one |

## Basic Usage

```bash
# Show help
./actor_iac.java --help

# Show help for a specific command
./actor_iac.java run --help
```
