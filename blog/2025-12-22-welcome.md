## What We Offer

### POJO-actor

A lightweight actor model library for Java. Turn any Plain Old Java Object into a concurrent actor without complex frameworks or boilerplate.

- Zero dependencies
- Type-safe messaging
- Built-in workflow support

### actor-IaC

Infrastructure as Code powered by actors. Define infrastructure workflows in simple YAML with state machine semantics.

- Parallel execution across nodes
- Built-in H2 database logging
- AI-friendly minimal syntax
